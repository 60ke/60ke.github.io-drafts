无参函数调用

先写一个rust的lib项目生成动态库

  • Cargo.toml
  • lib.rs
  • 编译
    运行 cargo build --release得到libembed.dylib

将文件按照如下放置

  • interface.h
  • main.c
  • 编译运行C
    gcc main.c -L. -lembed && ./a.out
    PS:linux下编译gcc main.c -L. -lembed -Wl,-rpath=./
    -Wl,-rpath=./用于指定程序运行寻找库的路径
    运行结果
  • test.go go run test.go结果与C运行一致

有参函数调用

  • lib.rs cargo build --release --out-dir . -Z unstable-options
